Mini-projeto de laboratório criado com o objetivo de praticar integração com APIs externas utilizando Laravel.

O projeto utiliza a PokeAPI, uma API pública e simples, como base inicial de aprendizado, permitindo foco em conceitos fundamentais como consumo de API, tratamento de erros, cache e organização do código.

A intenção é que este projeto sirva como base evolutiva, possibilitando, em um futuro próximo, a aplicação dos mesmos conceitos em APIs mais completas e complexas, com autenticação, rate limit e outros desafios reais.

Tenha em mente que esse projeto é desenvolvido nas horas vagas que possuo no trabalho/escola, ocasionando em commits de certa forma "inconsistentes".
